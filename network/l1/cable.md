# 通信ケーブル

- 銅線
  - ツイストペアケーブル
    - BASE-T や BASE-TX
      - T はツイスト
    - 8 本の銅線を 2 本ずつ撚り合わせさらに一つに束ねてケーブルにしている
    - ケーブル部分をアルミ箔でシールド処理している STP(Shielded Twist Pair)とシールド処理していない UTP(Unshielded Twist Pair)がある
    - LAN ケーブルは UTP が多い
  - 同軸ケーブル
- 光ファイバ

  - シングルモード光ファイバ
  - マルチモード光ファイバ
    - GI(Graded Index)型
    - SI(Step Index)型

- 光ファイバの方が早く,信号も減衰しにくく，電磁ノイズの影響も少なく，伝送距離も長いが，コストや取り回しが悪い
