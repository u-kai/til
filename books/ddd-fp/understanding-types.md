## 値と Objects の違い

- 値は単なる型の集まり
- 関数も値

  - int -> int という値

- 値はイミュータブル
  - 1 はずっと 1
- 値はただのデータなので振る舞いを持たない
- Objects はカプセル化されているデータ構造と，それに関連する振る舞い
- objects は状態をもち，ミュータブルで，全ての変更に関しては提供されている振る舞いを経由する
- 関数型では value を変数や object よりも使うこと
